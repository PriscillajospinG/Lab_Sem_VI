# Experiment 6: Recording of Notepad Automation

## Aim
To automate the notepad with recording using UiPath.

## Description
Robotic Process Automation (RPA) automates repetitive tasks by mimicking human actions. In this experiment, UiPath Studio is used to automate the Notepad application using the recording feature, which captures actions like typing text and saving files. The recorded workflow is executed to automatically create, edit, and save a Notepad file, demonstrating how UiPath simplifies basic desktop automation.

## Procedure

### Step 1: Start New Project
- Open UiPath Studio
- Click **Process** to start a new project

### Step 2: Create Blank Process
- Create a new blank process by giving a name
- Click **Create**

### Step 3: Open Main Workflow
- Click **Open Main Workflow**

### Step 4: View Main Workflow
- You will see the main workflow area

### Step 5: Open Notepad Application
- Open the Notepad application on your computer

### Step 6: Start App/Web Recording
- From the **Design** ribbon, click on **App/Web Recorder**

### Step 7: Select Notepad Window
- A recording panel will appear
- Select the Notepad application by clicking **Indicate Window**
- Click on the Notepad window to select it

### Step 8: Type Text in Notepad
- A dialog box appears asking for text to enter
- Type the text you want to enter in Notepad (e.g., "RPA Recording Demo")
- The recording captures this as a Type Into action

### Step 9: Click on File Menu
- To select File menu, go to **Click** option in the recorder
- Choose **File** from the Notepad menu bar

### Step 10: Click Save As
- Choose **Click** in the recorder
- Press **F2** to add a delay (gives you 3 seconds to navigate)
- Then click on **Save As** option

### Step 11: Enter File Name
- A dialog box appears to enter the file name
- Type the desired filename (e.g., "RecordedFile")

### Step 12: Save the File
- Use **Click** to select the **Save** button

### Step 13: Return to UiPath Workspace
- After recording all actions, click **Save & Exit** on the recorder
- UiPath workspace will open with the recorded workflow

### Step 14: Run the Automation
- Click **Run** or press **F5** to execute the recorded workflow

## Workflow Structure (Auto-generated by Recording)
```
Main.xaml
├── Use Application/Browser (Notepad)
│   ├── Type Into (text content)
│   ├── Click (File menu)
│   ├── Click (Save As)
│   ├── Type Into (filename)
│   └── Click (Save button)
```

## Output
- The recorded automation plays back automatically
- Notepad opens (if not already open)
- Text is typed into Notepad
- File menu is clicked
- Save As dialog opens
- File is saved with the specified name

## Recording vs Manual Workflow Creation

| Aspect | Recording | Manual Creation |
|--------|-----------|-----------------|
| **Speed** | Faster to create | Takes more time |
| **Accuracy** | May need refinement | More precise control |
| **Learning Curve** | Easier for beginners | Requires more knowledge |
| **Flexibility** | Limited customization | Full customization |

## Tips for Better Recording
- ✅ Close unnecessary applications before recording
- ✅ Use **F2** key to add delays when needed
- ✅ Keep mouse movements steady
- ✅ Review and edit recorded activities if needed
- ❌ Avoid fast clicking during recording
- ❌ Don't switch between multiple windows unnecessarily

## Result
Thus, the recording of notepad automation is completed successfully.
