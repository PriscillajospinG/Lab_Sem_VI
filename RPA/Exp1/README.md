# Experiment 1: Study of RPA Developer Role and Displaying the Welcome Message

## Aim
To study the role of an RPA Developer and understand the UiPath Platform components (Studio, Orchestrator, and Robot) by creating a simple automation workflow that displays a Welcome Message.

## Description
This experiment introduces the fundamentals of Robotic Process Automation (RPA) and the responsibilities of an RPA Developer. RPA is a technology that allows software robots to mimic human actions on digital systems such as opening applications, entering data, clicking buttons, and making rule-based decisions. It is mainly used for processes that are manual, repetitive, rule-based, and have minimal exceptions.

### UiPath Platform Components
| Component | Description |
|-----------|-------------|
| **UiPath Studio** | For designing automation workflows visually |
| **UiPath Orchestrator** | For managing, monitoring, and controlling robots |
| **UiPath Robot** | For executing the automation process (Attended or Unattended) |

## Procedure

### Part 1: Installing UiPath

#### Step 1: Go to UiPath Website
- Open your browser and go to [uipath.com](https://www.uipath.com)

#### Step 2: Access Free Trial
- In UiPath website, click on **Try UiPath Free**

#### Step 3: Sign Up
- Sign up using your email (kmail)

#### Step 4: Verify Email
- After sign up, verify through your email
- It will redirect you to the UiPath dashboard

#### Step 5: Download UiPath Studio
- Click **Download UiPath Studio**
- The UiPath Studio setup execution file will download

#### Step 6: Run the Installer
- Open the `.exe` file
- It will take a few minutes to open

#### Step 7: Select UiPath Studio Pro
- Select **UiPath Studio Pro**
- Wait for the installation to complete

#### Step 8: Create a New Process
- To work on a project, go to **Process**
- Create a process by giving a name
- Click **Open Main Workflow**

#### Step 9: UiPath Studio Workflow
- This opens the UiPath Studio Workflow interface

---

### Part 2: Displaying a Welcome Message

#### Step 1: Start New Project
- Open UiPath Studio
- Click **Process** to start a new project

#### Step 2: Create Blank Process
- Create a new blank process by giving a name
- Click **Create**

#### Step 3: Open Main Workflow
- Click **Open Main Workflow**

#### Step 4: View Main Workflow
- You will see the main workflow area

#### Step 5: Add Message Box Activity
- In **Activities** panel, search for **Message Box**
- Click and drag it to drop in the activity area

#### Step 6: Enter Welcome Message
- Type the message text inside double quotes:
  ```
  "Welcome to RPA Lab!"
  ```

#### Step 7: Run the Sequence
- Click **Run** or press **F5** to execute the workflow

## Output
A message box appears displaying: **Welcome to RPA Lab!**

## Result
Thus, the role of an RPA Developer and the components of the UiPath Platform were studied successfully, and a simple automation workflow was created and executed to display the Welcome Message successfully.
